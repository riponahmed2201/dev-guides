# 6. Bit Manipulation Basics

কম্পিউটার মূলত ১ এবং ০ (Binary) ছাড়া কিছুই বোঝে না। বিট ম্যানিপুলেশন হলো সংখ্যার এই বাইনারি লেভেলে সরাসরি কাজ করার একটি অত্যন্ত দ্রুত এবং কার্যকর পদ্ধতি।

## বাইনারি রিপ্রেজেন্টেশন (Binary Representation)

প্রতিটি পূর্ণসংখ্যাকে বাইনারি বা ২-ভিত্তি বিশিষ্ট সংখ্যায় প্রকাশ করা যায়।

- **উদাহরণ:** ৫ এর বাইনারি হলো `101` (2^2 _ 1 + 2^1 _ 0 + 2^0 \* 1)।

## দশমিক থেকে বাইনারি রূপান্তর (Decimal to Binary Conversion)

একটি দশমিক সংখ্যাকে বার বার ২ দিয়ে ভাগ করে এবং ভাগশেষগুলো (Remainder) উল্টো করে সাজিয়ে বাইনারি পাওয়া যায়।

## বিটওয়াইজ অপারেশন (Bitwise Operations)

| Operator | Name | Logic                                            |
| :------- | :--- | :----------------------------------------------- | -------------------------------- |
| `&`      | AND  | দুটির মধ্যে দুটি বিট ১ হলে রেজাল্ট ১, অন্যথায় ০। |
| `        | `    | OR                                               | যেকোনো একটি বিট ১ হলে রেজাল্ট ১। |
| `^`      | XOR  | দুটি ভিন্ন বিট হলে ১, একই হলে ০।                 |
| `~`      | NOT  | বিটকে উল্টে দেয় (১ থাকলে ০, ০ থাকলে ১)।          |

## বিট শিফটিং (Bit Shifting)

### Left Shift (<<)

বিটগুলোকে বাম দিকে সরিয়ে দেয় এবং ডানে শূন্য যোগ করে। এটি সংখ্যাকে ২ দিয়ে গুণ করার সমান।

- `x << n` মানে x \* 2^n

### Right Shift (>>)

বিটগুলোকে ডান দিকে সরিয়ে দেয়। এটি সংখ্যাকে ২ দিয়ে ভাগ করার সমান।

- `x >> n` মানে x / 2^n

## গুরুত্বপূর্ণ ট্রিকস (Common Tricks)

### ১. একটি সংখ্যা ২ এর পাওয়ার কিনা চেক করা

যদি `(n & (n-1)) == 0` হয় এবং n > 0 হয়, তবে সংখ্যাটি ২ এর পাওয়ার (2^k)।

### ২. সেট বিট (Count Set Bits) কাউন্ট করা

একটি সংখ্যার বাইনারি রিপ্রেজেন্টেশনে কতগুলো ১ আছে তা বের করা।

- **প্রবাদ:** Brian Kernighan's Algorithm ব্যবহার করে এটি খুব দ্রুত করা যায়।

### ৩. বিজোড়-জোড় বের করা

`n & 1` যদি ১ হয় তবে বিজোড়, ০ হলে জোড়।

---

> [!IMPORTANT]
> বিট ম্যানিপুলেশন ব্যবহার করলে কোড অনেক সময় O(1) এ সমাধান করা সম্ভব হয় যা অন্যভাবে করতে O(n) বা O(log n) লাগত।
