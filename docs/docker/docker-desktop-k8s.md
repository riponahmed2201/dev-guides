# Docker Desktop Kubernetes (ডকার ডেস্কটপ কুবারনেটিস)

আপনি যদি কুবারনেটিস শিখতে চান বা লোকাল মেশিনে কুবারনেটিস ক্লাস্টার টেস্ট করতে চান, তবে আলাদা কোনো টুল ছাড়াই 'Docker Desktop' ব্যবহার করে এটি করা সম্ভব। এটি ওয়ান-ক্লিক কুবারনেটিস ক্লাস্টার সেটআপ করার সুবিধা দেয়।

## ১. কুবারনেটিস এনাবল করা (Enable Kubernetes)

ডকার ডেস্কটপে কুবারনেটিস চালু করতে নিচের ধাপগুলো অনুসরণ করুন:
1. ডকার ডেস্কটপ ওপেন করুন এবং **Settings** (গিয়ার আইকন) এ ক্লিক করুন।
2. বাম পাশের মেনু থেকে **Kubernetes** সিলেক্ট করুন।
3. **Enable Kubernetes** চেকবক্সে টিক দিন।
4. **Apply & Restart** বাটনে ক্লিক করুন। 

ডকার প্রয়োজনীয় ইমেজগুলো ডাউনলোড করবে এবং লোকাল কুবারনেটিস ক্লাস্টার চালু করবে। স্ক্রিনের নিচের দিকে কুবারনেটিস আইকন সবুজ হলে বুঝবেন ক্লাস্টার রেডি।

---

## ২. kubectl বেসিকস (kubectl basics)

কুবারনেটিস ক্লাস্টার ম্যানেজ করার জন্য কমান্ড লাইন টুল হলো `kubectl`। ডকার ডেস্কটপে এটি অটোমেটিক ইন্সটল হয়ে যায়।

### ক্লাস্টারের অবস্থা চেক করা:
```bash
kubectl cluster-info
kubectl get nodes
```

---

## ৩. ডকার ইমেজ থেকে পড (Pod) তৈরি

কুবারনেটিসের ক্ষুদ্রতম ইউনিট হলো 'Pod'। আপনি আপনার ডকার ইমেজ ব্যবহার করে সরাসরি পড তৈরি করতে পারেন।

```bash
# একটি এনজিনক্স পড তৈরি
kubectl run my-nginx --image=nginx:alpine
```

### পড লিস্ট দেখা:
```bash
kubectl get pods
```

---

## ৪. সার্ভিস এক্সপোজ করা (Service Exposure)

পড তৈরি করলেই সেটি লোকাল মেশিনের ব্রাউজারে দেখা যায় না। এজন্য সার্ভিস এক্সপোজ করতে হয়।

```bash
# পোর্ট ফরওয়ার্ডিং বা এক্সপোজ কমান্ড
kubectl expose pod my-nginx --port=80 --type=NodePort
```

অথবা সরাসরি লোকাল পোর্টে ম্যাপ করতে:
```bash
kubectl port-forward pod/my-nginx 8080:80
```
এখন ব্রাউজারে `localhost:8080` ভিজিট করলে এনজিনক্স পেজটি দেখতে পাবেন।

---

## ৫. লোকাল ডেভেলপমেন্ট ফ্লো (Local Development)

লোকাল কুবারনেটিস ব্যবহারের আসল মজা হলো আপনি আপনার লোকাল ডকার ইমেজ ব্যবহার করতে পারেন।
1. প্রথমে ডকার ইমেজ বিল্ড করুন: `docker build -t my-app:v1 .`
2. এরপর কুবারনেটিস ম্যানিফেস্টে সেই ইমেজটি ব্যবহার করুন।
3. `kubectl apply -f deployment.yaml` ব্যবহার করে ডিপ্লয় করুন।

> [!TIP]
> ডকার ডেস্কটপের কুবারনেটিস এবং আপনার লোকাল ডকার একই ইমেজ স্টোরেজ শেয়ার করে, তাই আপনাকে ইমেজ পুশ করতে হয় না।
