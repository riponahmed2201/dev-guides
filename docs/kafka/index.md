# Apache Kafka (অ্যাপাচি কাফকা)

অ্যাপাচি কাফকা বর্তমানে এন্টারপ্রাইজ স্ট্রিমিং এবং মেসেজিংয়ের জন্য বিশ্বের অন্যতম সেরা ওপেন-সোর্স টুল। ফরচুন ১০০ কোম্পানির ৮০% এরও বেশি কোম্পানি কাফকার ওপর ভরসা করে।

এই নেক্সট-জেনারেশন বিগ ডেটা মেসেজিং সিস্টেমটি Twitter, LinkedIn, Netflix, Uber-এর মতো কোম্পানিগুলোর হয়ে প্রতিদিন বিলিয়ন বিলিয়ন মেসেজ হ্যান্ডেল করে। সম্প্রতি এটি প্রতিদিন ১.১ ট্রিলিয়ন মেসেজ হ্যান্ডেল করার মাইলফলক স্পর্শ করেছে।

এই কোর্সে আপনি কাফকা শিখে রিয়েল-টাইম অ্যানালিটিক্স ক্যারিয়ারে অন্যদের থেকে এগিয়ে থাকতে পারবেন।
