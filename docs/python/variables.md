# Variables & Data Types (ভেরিয়েবল এবং ডেটা টাইপ)

Python এ কোনো ডেটা সেভ করে রাখার জন্য আমরা **Variables** ব্যবহার করি। এটি একটি বক্সের মতো যেখানে আপনি আপনার প্রয়োজনীয় তথ্য জমা রাখতে পারেন।

---

## ১. Variables Declaration

অন্যান্য ল্যাঙ্গুয়েজের মতো Python এ ভেরিয়েবল ডিক্লেয়ার করার জন্য কোনো কি-ওয়ার্ড (যেমন: `int`, `var`, `let`) লাগে না। শুধু নাম এবং ভ্যালু দিলেই হয়।

```python
name = "Ripon"
age = 25
is_student = True
```

### naming Rules (নামকরণের নিয়ম):
- নাম অবশ্যই লেটার (a-z) বা আন্ডারস্কোর (`_`) দিয়ে শুরু হতে হবে।
- নাম্বার দিয়ে শুরু করা যাবে না (যেমন: `1name` ভুল)।
- শুধুমাত্র অক্ষর, সংখ্যা এবং আন্ডারস্কোর থাকতে পারে।
- Python একটি **Case-sensitive** ল্যাঙ্গুয়েজ (অর্থাৎ `Name` এবং `name` আলাদা)।

---

## ২. Data Types (ডেটা টাইপ)

Python এ প্রধানত ৪টি বেসিক ডেটা টাইপ আছে:

| Type | Description | Example |
| :--- | :--- | :--- |
| **int** | পূর্ণসংখ্যা (Integer) | `10`, `-5`, `1000` |
| **float** | দশমিক সংখ্যা | `10.5`, `3.1416` |
| **str** | টেক্সট বা স্ট্রিং | `"Hello"`, 'Bangla' |
| **bool** | সত্য বা মিথ্যা | `True`, `False` |

---

## ৩. Type Casting (টাইপ পরিবর্তন)

কখনো কখনো এক টাইপ থেকে অন্য টাইপে ডেটা নেওয়ার প্রয়োজন হয়। একে **Type Casting** বলে।

```python
x = "10"       # এটি একটি String
y = int(x)     # এখন এটি Integer হয়ে গেল
z = float(y)   # এটি এখন 10.0 (Float)
```

---

## ৪. Multiple Assignment

একসাথে অনেকগুলো ভেরিয়েবলে ভ্যালু এসাইন করা যায়।

```python
a, b, c = 1, 2, 3
x = y = z = 0  # তিনটির ভ্যালুই ০ হবে
```

::: tip মনে রাখবেন
Python হলো একটি **Dynamically Typed** ল্যাঙ্গুয়েজ। এর মানে হলো আপনাকে আগে থেকে টাইপ বলে দিতে হয় না, ভ্যালু দেখেই Python বুঝে নেয় এটি কোন টাইপের ডেটা।
:::
