# Python Basics (প্রাথমিক ধারণা)

Python শেখার যাত্রা শুরু করার জন্য এই অংশটি সবচেয়ে গুরুত্বপূর্ণ। এখানে আমরা একদম শুরু থেকে সবকিছু সহজভাবে বুঝবো।

---

## ১. Python কি এবং কেন শিখবে?

**Python** হলো একটি হাই-লেভেল, ইন্টারপ্রেটেড প্রোগ্রামিং ল্যাঙ্গুয়েজ যা বর্তমানে বিশ্বের সবচেয়ে জনপ্রিয় ভাষাগুলোর একটি।

### কেন শিখবে?
- **সহজ সিনট্যাক্স (Simple Syntax):** Python এর কোড পড়া সাধারণ ইংরেজি পড়ার মতো সহজ। 
- **ভার্সেটাইল (Versatile):** এটি দিয়ে ওয়েব ডেভেলপমেন্ট, ডেটা সায়েন্স, আর্টিফিশিয়াল ইন্টেলিজেন্স, অটোমেশন — সবকিছুই করা সম্ভব।
- **বিশাল কমিউনিটি:** যেকোনো সমস্যায় ইন্টারনেটে প্রচুর সাহায্য পাওয়া যায়।

---

## ২. Installation (সেটআপ)

কোডিং শুরু করার আগে আপনার কম্পিউটারে Python এবং একটি Code Editor (যেখানে কোড লিখবেন) থাকতে হবে।

### Python ইন্সটল করা:
1. [python.org](https://www.python.org/downloads/) এ যান।
2. আপনার OS (Windows/Mac) অনুযায়ী লেটেস্ট ভার্সনটি ডাউনলোড করুন।
3. ইন্সটল করার সময় অবশ্যই **"Add Python to PATH"** বক্সে টিক দিয়ে নিবেন।

### Editor সেটআপ (VS Code):
আমরা সাধারণত **Visual Studio Code (VS Code)** ব্যবহার করি কারণ এটি হালকা এবং অনেক কাজের।
- VS Code ইন্সটল করার পর `Extensions` ট্যাব থেকে **"Python"** প্লাগইনটি ইন্সটল করে নিন।

## ৩. First Program: Hello World

প্রথা মেনেই আমাদের প্রথম কোড হবে স্ক্রিনে "Hello World" প্রিন্ট করা।

```python
print("Hello World")
```

### এই কোডটি কী করছে?
- `print()` হলো Python এর একটি বিল্ট-ইন ফাংশন যা ব্র্যাকেটের ভেতরের লেখাটিকে আপনার স্ক্রিনে দেখায়।
- লেখাটিকে ডাবল কোটেশন (`" "`) এর ভেতরে রাখা হয়েছে কারণ এটি একটি **String** বা টেক্সট।

## ৪. Python Interpreter এবং REPL

### Interpreter (ইন্টারপ্রেটার):
Python হলো একটি ইন্টারপ্রেটেড ল্যাঙ্গুয়েজ। এর মানে হলো এটি কোডকে লাইন-বাই-লাইন পড়ে এবং সাথে সাথে রান করে। 

### REPL:
REPL মানে হলো **Read-Eval-Print Loop**। আপনি যদি আপনার টার্মিনালে শুধু `python` লিখে এন্টার দেন, তবে একটি ইন্টারঅ্যাক্টিভ শেল শুরু হবে। সেখানে আপনি সরাসরি কোড লিখে সাথে সাথে আউটপুট দেখতে পাবেন।

## ৫. Comments এবং Documentation

কোড কেন লিখেছেন বা কী কাজ করছে তা নিজের বা অন্যদের বোঝার জন্য **Comments** ব্যবহার করা হয়। Python ইন্টারপ্রেটার কমেন্ট করা লাইনগুলো এড়িয়ে যায়।

### Single Line Comment:
এর জন্য `#` ব্যবহার করা হয়।
```python
# এটি একটি কমেন্ট, এটি রান হবে না
print("Hello")
```

### Documentation Strings (Docstrings):
বড় কোনো টেক্সট বা ডকুমেন্টেশনের জন্য ট্রিপল কোটেশন (`""" """`) ব্যবহার করা হয়। এটি সাধারণত ফাংশন বা ক্লাসের শুরুতে ডেসক্রিপশন হিসেবে থাকে।
```python
"""
এই প্রোগ্রামটি পাইথন বেসিক্স 
শেখানোর জন্য তৈরি করা হয়েছে।
"""
print("Docstrings are cool!")
```
