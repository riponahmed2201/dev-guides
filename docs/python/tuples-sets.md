# Tuples & Sets

দুটিই কালেকশন কিন্তু এদের কাজ আলাদা।

---

## ১. Tuples (টুপল)
টুপল তৈরি করা হয় ফার্স্ট ব্র্যাকেট `( )` দিয়ে। এর প্রধান বৈশিষ্ট্য হলো এটি **Immutable** (পরিবর্তন করা যায় না)।

```python
point = (10, 20)
# point[0] = 15  # এটি এরর দিবে
```

**কেন ব্যবহার করবেন?**
- যখন আপনি চান কোনো ডেটা প্রোগ্রাম চলাকালীন কেউ পরিবর্তন করতে না পারে (যেমন: Database Credential বা Coordinates)।

---

## ২. Sets (সেট)
সেট তৈরি করা হয় সেকেন্ড ব্র্যাকেট `{ }` দিয়ে। সেটে কোনো ডুপ্লিকেট ভ্যালু থাকে না এবং এর কোনো নির্দিষ্ট ক্রম বা ইনডেক্স নেই।

```python
numbers = {1, 2, 2, 3, 4}
print(numbers) # Output: {1, 2, 3, 4} (ডুপ্লিকেট ২ একটিই দেখাবে)
```

**প্রধান অপারেশন:**
- **Union**: দুটি সেটকে একত্র করা।
- **Intersection**: দুটির মধ্যে কমন ভ্যালু বের করা।
- **Difference**: এক সেট থেকে অন্য সেটের ভ্যালু বাদ দেওয়া।
